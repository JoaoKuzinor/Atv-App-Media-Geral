# App Média Geral

Aplicativo Android desenvolvido em **Kotlin** utilizando **Jetpack Compose**, com o objetivo de calcular a média geral de um aluno a partir de três notas informadas pelo usuário.

## Autor
**João Pedro Kuzinor de Lima**

## Objetivo
O aplicativo permite inserir o nome do aluno e três notas (TP1, TP2 e TP3). Após o cálculo, é apresentada a média final e o status do aluno, considerando o seguinte critério:

- Média maior ou igual a 6: Aprovado
- Média menor que 6: Reprovado

## Tecnologias Utilizadas
- Kotlin
- Jetpack Compose
- Android Studio

## Funcionalidades
- Cálculo de média com duas casas decimais
- Validação de notas entre 0 e 10
- Bloqueio da operação caso o nome não seja informado
- Limpeza automática dos campos após o cálculo
- Interface construída com componentes visuais do Material Design

## Como Executar o Projeto
1. Abra o projeto no **Android Studio**
2. Execute em um emulador Android ou dispositivo físico
3. Clique no botão **Run**
